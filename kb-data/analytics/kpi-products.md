# Product KPIs

Tracks product catalog health using PBI Product Measures. Covers active SKU count, roaster capacity utilization (current week and historical), inventory levels (available, reserved, true available), order pipeline, and per-SKU margin metrics.

## Data Notes

- **Active PIM SKUs** counts products with active status in the Product Information Management system (DimProduct).
- **Capacity %** measures roaster production utilization. Current week and historical variants allow trend analysis. Values above 100% indicate over-allocation.
- **True Available Inventory** = Available Inventory - Reserved Inventory. This is the actual fulfillable stock.
- **Reserved to Ship** vs **Reserved Shipped**: "to ship" is committed but not yet dispatched; "shipped" has left the warehouse.
- **SKU Margin** is the average margin at the individual SKU level, distinct from order-level margins in [[kpi-revenue]].

## PBI Measures

This doc draws from the **Product Measures** folder in BeanzCoreDatabricks:

- Active PIM SKUs Count
- % of Capacity (total, Current Week, Historical)
- Sum of Available / Reserved / True Available Inventory, Inventory Level
- New Orders, Reserved Order Shipped, Reserved to be Shipped
- Current Average Sku Margin $ / %
