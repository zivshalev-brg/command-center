# Revenue KPIs

Tracks financial performance across all beanz.com markets using PBI Order Measures. Covers actual revenue, subscription revenue split, gross/net sales, margin metrics at order and SKU level, shipping costs, and unit volumes in both count and kilogram terms.

## Data Notes

- **Actual Revenue** uses the `_OrderRevenueAmount` column from FactOrder, which is net of discounts and refunds.
- **Subscription Revenue %** is calculated as `Actual Subscription Revenue / Actual Revenue`. A rising percentage indicates healthy subscription mix shift.
- **Gross Margin %** and **Order Margin %** differ: gross margin excludes shipping; order margin includes it when using the `_incl_shipping` variants.
- **Units (KG)** converts bag counts to weight using the SKU weight attribute from DimProduct. Free and paid units are split separately.
- **AOV** (Average Order Value) divides total order value by completed order count. **ABV** (Average Basket Value) and **ABS** (Average Basket Size) provide complementary basket-level views.

## PBI Measures

This doc draws from the **Order Measures** folder in BeanzCoreDatabricks:

- Actual Revenue, Actual Revenue LY, Actual Revenue % Change from LY
- Actual Subscription Revenue, Subscription Revenue % of Actual Revenue
- Gross Sales for Completed Orders, Net Sales After Discounts for Completed Orders
- Order Value for Completed Orders (+ LY), Order Discount for Completed Orders
- Average Order Value, ABV, ABS, ASP (+ incl Shipping)
- Count of Completed Orders (+ LY), Count of All Orders
- Current Average Gross Margin % / $, Current Average Order Margin % / $ (+ incl Shipping)
- Total Order Margin $ / % (+ LY), Order Sku Margin $ / %
- Actual Cost to BRG, Total Shipping Cost, Average Shipping Cost
- Sum of Units (+ LY, % Change, KG, Free, Paid)
- Average Cost per KG, Avg RRP per KG, Avg KG per Subscription
- Count of Completed Subscription / NonSubscription Orders
- Average Order Frequency by Subscriptions
